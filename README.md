# Palmtree Curl

[![License](http://img.shields.io/packagist/l/palmtree/curl.svg)](LICENSE)

A curl class to make http requests a bit easier.

## Requirements
* PHP >= 5.6

## Installation

Use composer to add the package to your dependencies:
```bash
composer require palmtree/curl
```
